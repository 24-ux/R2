# 底盘模式命令表（不含活动电机）

适用场景：模块使能 `data[0]=0x01`（底盘模式），并且**不下发/不关心活动电机**（等价于 `data[1]` 的 `bit0=0`）。

## 1. 帧与校验

约定其他字节均为 0：
- `data[0] = 0x01`
- `data[1] = 待查值`
- `data[2] ~ data[19] = 0x00`

校验：
- `checksum = (sum(data[0..19]) & 0xFF) = (0x01 + data[1]) & 0xFF`

完整帧可按如下模板拼接：
- `AA 55 01 {data[1]} 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 {checksum} 0D 0A`

## 2. `data[1]` 位定义（bit0 固定为 0）

`data[1] = speed( bit7~6 ) + move( bit5~3 ) + rot( bit2~1 ) + flex(bit0=0)`

- 速度档（bit7~6）
  - `01` 低速
  - `10` 中速
  - `11` 高速
- 平移方向（bit5~3）
  - `000` 不平移（NONE）
  - `001` 前进（FORWARD）
  - `010` 后退（BACKWARD）
  - `011` 左移（LEFT）
  - `100` 右移（RIGHT）
- 旋转方向（bit2~1）
  - `00` 不旋转
  - `01` 左旋
  - `10` 右旋
- 活动电机（bit0）
  - 固定 `0`（不含活动电机）
